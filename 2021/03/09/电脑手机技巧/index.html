电脑手机技巧

QQNT版占用内存高

点一下左边的#（QQ频道），再关掉就可以了

npm下载速度慢

命令行输入npm config set registry https://registry.npmmirror.com

删除文件夹是提示0x80070091目录不是空的

复制一个文件进去后再删除

下载优酷视频

安装python后pip3 install you-get
you-get 视频地址

联通手机把积分换成话费

发JFJF#Q到10010

小米手机静音看视频直播

对着小爱同学说实时字幕，再打开视频就可以看到悬浮框实时识别的字幕了

calibre直接打开epub文件后会修改文件导致sha1校验变化

打开一本书，右键单机右侧，首选项，杂项，保存一份标注/书签在电子书文件中便于共享，去掉勾

chrome提示网站不是私密链接

chrome右键属性，目标后面引号外添加–test-type –ignore-certificate-errors

poweramp蓝牙播放音乐卡顿

在设置，音频，输出，尝试切换成其他输入

excel设置正数是红色，负数是绿色

选单元格，可以按住ctrl多选，设置单元格格式，自定义，输入[红色]#,##0.00;[绿色]-#,##0.00

inoreader访问不了

网页端inoreader改成innoreader

安卓端进入设置，随便一个界面，摇晃手机，inoreader改成innoreader

winrar下载地址

https://www.win-rar.com/fileadmin/winrar-versions/sc/sc20231013/rrlb/winrar-x64-624sc.exe
https://www.win-rar.com/fileadmin/winrar-versions/sc/sc20231013/wrr/winrar-x64-624sc.exe

github pages设置

要在文件夹内新建一个文件CNAME，里面是你的域名

萤石摄像头打开rtsp，可用播放器播放，关闭红外

打开手机萤石云视频app，我的，工具，局域网设备预览，开始扫描

选择设备，输入密码是6位大写英文，在摄像头下面贴纸上可以找到

右上角设置，更多设置，本地服务开关，RTSP打开，保存，有个设备可能没这个设置

关闭红外：上面第三步后，右上角设置，日夜切换，强制白天

电脑播放可以使用vlc，potplayer或vlc，安卓手机播放，使用vlc，新建my.m3u8

foobar2000歌词插件、搜索、布局整合

插件ESLyric，下载地址https://github.com/ESLyric/release/releases

建立自己的布局，以左边播放列表右边歌词为例

view-layout-create scratchbox，点击灰色的地方spliter(left/right)

点击左边add new ui element，playlist tabs，再点左边，playlist view

点击右边add new ui element，ESLyric

点击窗口左上edit， apply layout就成功了，重启foobar2000后就自动加载歌词了
