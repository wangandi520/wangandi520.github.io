舒适阅读电子书和漫画

发表于 2021-03-16 更新于 2024-01-15 分类于 评测 阅读次数： 本文字数： 3745 阅读时长 ≈ 3 分钟

更新日期

2023.12.7

设备

电纸书

  • kindle：封闭系统，边框宽，只适合看亚马逊的书，没其他需求的朋友。不推荐。
  • 文石boox：安卓系统，系统bug多更新慢，遇到过半年修复一个bug。适合安装app阅读，或看部分pdf，轻度推荐。具体评测：nova2，nova pro，poke pro，
  • 海信：安卓系统，5寸左右的手机，方便随时阅读，看小说的话轻度推荐。

看纯文字书，推荐国产安卓电纸书，开放安卓系统。看漫画至少8寸，但是很多残影问题处理不好。

看漫画推荐安卓平板+perfect viewer或ipad+comicshare，比用电纸书舒服。

eink设备建议满足这些条件：

  • 300ppi
  • 开放系统
  • 最新版屏幕（如carta1300）
  • 重量轻

辅助设备

  • usb脚踩翻页键：可以在驱动里设置成Page down或其他按键。
  • 躺椅：靠背角度可调，看累了可以直接躺下睡觉。
  • 其他：蓝牙手柄，蓝牙鼠标（有侧键），平板支架（可调角度），线控耳机，宜家TERTIAL特提亚台灯，护眼灯泡，查看灯光频闪，使用手机摄像专业模式，快门调成1/4000。

我的设备

安卓平板+tf卡。之前联想y700最新系统zui15更新了一坨，任务栏关不了，有些app后退要滑两下，很影响使用，不推荐购买使用，可以线刷退回到zui13使用，参照https://andi.wang/2024/01/13/%E6%8B%AF%E6%95%91%E8%80%85y700%E4%B8%80%E4%BB%A3%EF%BC%882022%EF%BC%89%E5%B9%B3%E6%9D%BF%E5%88%B7%E6%9C%BA%E8%BF%87%E7%A8%8B/。
看漫画：用X-plore，无线文件共享，通过电脑传漫画到tf卡里，perfect viewer阅读。
看书：使用resilio sync同步传书，静读天下专业版阅读，自建webdav同步进度、标注，标注导出成txt后使用python脚本转换，用hexo发布到自己的博客上。
看pdf：perfect viewer或者xodo。
看rss：inoreader。

软件：

安卓

  • 静读天下专业版：看本地电子书首选，支持格式多，azw3、epub、pdf都可以，排版也不错，部分划线方框格式不支持，多字体、弹出注释支持。8.4（804003）版设置里支持了电子墨水屏模式，去掉了动画，初步使用还不错。整理批注笔记，见下方个人小项目。多设备同步进度和批注，见搭建webdav服务器实现静读天下同步，可以使用蓝牙手柄翻页。总体推荐使用。Google play下载，推荐购买。
  • Perfect Viewer：看离线漫画首选，zip\rar\pdf都支持，支持分页，合页，功能多但偶尔有bug。可以在windows里设置共享，再在app右上角，书柜文件夹管理，添加局域网，填写电脑ip，就可以在设备上看电脑里的漫画。可以使用蓝牙手柄翻页，自定义每个按键。Google play下载。Bugs:perfect viewer，把正在读的书的压缩包里添加一个文件，新的文件显示不出来。samba联机阅读时，左上角转圈的时候退出来，程序会崩溃。主界面箭头下来，有的文件夹显示不出来。
  • ComicScreen：看离线漫画用，和Perfect Viewer比功能稍弱，速度快，自动裁切快且精准。
  • koreader：主要在电纸书上运行，功能多。操作稍微复杂、不人性化，不支持azw3。可以把每章结尾的脚注分到每一页下面，这一点很惊喜。下载地址https://github.com/koreader/koreader/releases
  • xodo：看pdf用，可以做笔记保存。
  • joplin：从浏览器收集文章，做笔记，可以自己搭建webdav同步，可加密。
  • X-plore：文件管理器，捐赠版可以通过wifi传输，电脑网页里上传，方便速度快。
  • 欧陆词典：查各种资料，支持mdx离线词典，网上很多资源。
  • QuickEdit：文本编辑器，查看个人笔记。
  • instapaper：从浏览器收集文章，稍后阅读。Google play下载。
  • 阅读3.0(legado)：主要在线追网文，本地epub排版不好。下载地址https://github.com/gedoor/legado/releases
  • 微信读书：不推荐。在线看书，书多，排版还行。和纸质书比有删减篡改。
  • kindle阅读：不推荐。功能少，边距大。
  • bilibili漫画：不推荐。有删减篡改。

下载安卓软件，https://4pda.to/，网上搜注册教程，需要根据俄文输入4位数字

ios

  • comicshare：看漫画用，支持局域网串流。

Windows

  • Neeview：电脑看离线漫画首选，支持压缩包、图片、pdf，支持分页，合页。查看历史记录、收藏夹很方便。可以一个功能绑定多个快捷键。可设备鼠标侧键开关放大镜后，离显示器稍远看。下载地址https://bitbucket.org/neelabo/neeview/downloads/，其他看漫画软件具体评测见对比几个离线漫画软件Honeyview,MangaMeeya,ComicsViewer,NeeView
  • HoneyView：看图看漫画，支持压缩包。
  • SumatraPDF：看pdf用，轻便。
  • Koodo-Reader：看epub电子书。
  • calibre：看epub电子书，转格式。下载地址https://calibre-ebook.com/。calibre直接阅读epub文件，每次会修改文件导致sha1变更，打开一本书，右键单机右侧，首选项，杂项，保存一份标注/书签在电子书文件中便于共享，去掉勾。
  • sigil：编辑epub文件。下载地址https://sigil-ebook.com/
  • goodsync：文件夹同步，一般是移动硬盘备份的时候用
  • 整理标注的个人python脚本：ClippingsToMarkdown，可以把文石Boox,Kindle,静读天下专业版的标注转换为Markdown格式，方便展示收藏，例子：读书笔记。
  • 整理漫画的个人python脚本：andyspythonscript, 用于改名，清理，整理漫画等。

下载windows软件，https://lrepacks.net/，https://rutracker.net/

跨平台同步

  • resilio sync：电脑和手机间同步或分享，功能强，推荐。
  • syncthing：同步或分享，开源，功能少操作复杂，有一些bug。

搭建linux服务器，可以给手机电脑同步用。参照使用archlinux搭建家庭数据服务中心

内容

文档格式

建议保存收藏的格式：电子书epub, azw3。漫画zip，rar。

搜索资源

  • zlibrary，大量书，，每天能下10本书，警惕百度搜到的是一些钓鱼网站，暂时能用的网址zh.1lib.sk
  • anna，zlibrary的镜像站，还有别的资源，不限量下载，但是速度很慢，网址zh.annas-archive.se
  • https://tool.yibook.org/，部分zlib书。superlib很多扫描版书，格式是zip的话解压缩（压缩包常见密码52gv，julian），转成图片或pdf用Pdg2Pic，图片文字锐化用ComicEnhancerPro，转成pdf用FreePic2Pdf。
  • https://zbook.lol/，部分zlib书。
  • www.manhuabudangbbs.com，大量高质量漫画，建议下载格式是zip或rar的

确定版本、译本、删减情况

先确定各个版本、删减、排版、插图清晰度情况等，可以在亚马逊每本书下面的评论、知乎、豆瓣、搜索引擎上找各种资料。

比如早期的白夜行（黑白大桥封面）有大量删减。新版(白色封面）少一些，无法确实是不是100%未删减。

外文书、古籍最首要看的译者的好坏

不要读非作者写的序

容易被误导，会有些人画蛇添足总结什么思想什么表达的，像是小学语文课一样无趣。

电子版还是纸质书

有些纸质书存在印刷质量差、翻译差、删减等问题，建议先看一遍电子版，觉得真是好书的话，再买纸质书收藏。

一些电子书平台会在纸质书正常出版的情况下，在电子版中额外删减一些内容。

读与记

可以同时读2，3本书，不至于枯燥一般也不会搞混。

好书只看一遍没看懂很正常，需要重复阅读。

读书笔记

读完一本书，建议在一个笔记中记录：书名、备注、阅读日期、评分。感想或是书评写在另一个笔记中。划线标注则每本书单独一个文件保存。设置成markdown格式，用微软的Visual Studio Code配合markdown插件，可以实时预览。也可以配合各种静态博客（像是hexo），方便发布到个人网站上或github pages。
