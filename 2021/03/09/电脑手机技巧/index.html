萤石摄像头打开rtsp，可用播放器播放，关闭红外

打开手机萤石云视频app，我的，工具，局域网设备预览，开始扫描

选择设备，输入密码是6位大写英文，在摄像头下面贴纸上可以找到

右上角设置，更多设置，本地服务开关，RTSP打开，保存，有个设备可能没这个设置

关闭红外：上面第三步后，右上角设置，日夜切换，强制白天

电脑播放可以使用vlc，potplayer或vlc，安卓手机播放，使用vlc，新建my.m3u8

foobar2000歌词插件、搜索、布局整合

插件ESLyric，下载地址https://github.com/ESLyric/release/releases

建立自己的布局，以左边播放列表右边歌词为例

view-layout-create scratchbox，点击灰色的地方spliter(left/right)

点击左边add new ui element，playlist tabs，再点左边，playlist view

点击右边add new ui element，ESLyric

点击窗口左上edit， apply layout就成功了，重启foobar2000后就自动加载歌词了
