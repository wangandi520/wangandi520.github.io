Windows和Android优质软件

发表于 2024-01-03 更新于 2024-03-04 分类于 学习笔记 阅读次数： 本文字数： 958 阅读时长 ≈ 1 分钟

建议只从各个软件官网下载。

两个平台

microsoft edge：浏览器，同步，从手机发网页到电脑都很方便。
microsoft todo：计划清单提醒。
microsoft onenote：做笔记，同步时快时慢。
joplin：做笔记，可以自己搭建服务器同步，可以导出，可以webdav同步。
resilio sync：同步文件用，只有两边都打开软件就自动完成。
localsend：跨平台发文件，发文字，简洁高效。

Windows

everything：搜索文件。
visual studio code：做笔记，写代码，支持预览markdown和别的插件，可以搭配resilio sync做跨平台同步笔记。
calibre：看电子书，编辑书，转换格式用。
mobaxterm：远程ssh。
pdfpatcher：处理pdf，转图片。
goodsync：备份文件。
酷我音乐：免费下载音乐。
foobar2000：离线听歌。可以自定义窗体结构。
potplayer：离线看视频，无广告。
qbittorrent：下载bt文件，速度慢的话要网上搜一些tracker添加进去。
neeview：看漫画，pdf，可以导出设置和历史记录。
honeyview：看图片。
sumatrapdf：看pdf。
autohotkey：自制一些脚本，简化操作，低门槛。
python：自制一些脚本，稍有一些门槛。
open broadcaster software：录制屏幕，或者直播。
total commander：管理文件，批量重命名。
goldendict：查词典，支持mdx。
reaper：做音乐，录音，练键盘，玩吉他软件效果器。
faststone capture：截图。
ditto：管理剪贴板。
xmind8：做思维导图。
http file server 2.3m：电脑临时做服务器，给其他设备传文件。

Android

静读天下专业版：离线看电子书，可以导出标注，可以多设备同步。
xodo：看pdf。
poweramp：离线听歌。
mx player：离线看视频。
feedme：看rss。
quickedit：编辑文本文件，可以搭配resilio sync做跨平台同步笔记。
x-plore：管理文件，wifi传文件。
欧陆词典：查词典，支持mdx。
andromoney：记账，可以导出。
gboard：输入法，无广告。
