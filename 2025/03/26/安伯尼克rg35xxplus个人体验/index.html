安伯尼克rg35xxplus个人体验

发表于 2025-03-26 更新于 2025-04-13 分类于 极客 阅读次数： 本文字数： 3339 阅读时长 ≈ 3 分钟

目录

安伯尼克rg35xxplus使用技巧，RetroArch使用技巧（跨平台）。
简单评价一下安伯尼克rg35xxplus，rg34xx，miyoo mini v4这三台掌机。

安伯尼克rg35xxplus简评

优点

  1. 手感舒服，勉强能单手操作，menu键容易按，位置也方便。
  2. 双tf卡，掌机多的，卡2换个机器就能继续玩，方便。

缺点或bug

  1. 关机再开机，系统时间总会差几分钟，没关机待机的时候也会差，测试大概每2小时会快1分钟。rg34xx没这个问题。
  2. 背面4个按键松松垮垮的。
  3. 每次退出游戏很慢，测试了下大概10秒，有点离谱了，进游戏才6秒。
  4. 玩游戏的时候电池盖轻微松动，不是很舒服。
  5. 快充的typec线没法充电，只能用低端的线。
  6. wifi性能差，隔了两面墙，连上wifi几乎要20秒。

安伯尼克rg34xx简评

优点

  1. 外形比较像gba吸引眼球。
  2. 双tf卡。

缺点或bug

  1. 对gba原装没有执念，感觉方向键太小了
  2. menu键在顶端按的姿势不太舒服，设置在掌机正面会方便很多。
  3. 买的冰川蓝，玩的时间长有点汗的时候，感觉外壳黏糊糊的。
  4. 每次退出游戏很慢，测试了下大概10秒，有点离谱了，进游戏才6秒。
  5. 玩游戏的时候电池盖轻微松动，不是很舒服。
  6. 快充的typec线没法充电，只能用低端的线。
  7. wifi性能差，隔了两面墙，连上wifi几乎要20秒。

安伯尼克rg34xxsp简评

优点

  1. 躺着单手玩，可以在转轴位置撑住机器。

缺点或bug

  1. 应用中心，apps，电源灯开关，设置关闭后，开屏状态充电，电源灯又会变成常亮状态。

miyoo mini v4简评

优点

  1. 手感好，很小，放裤兜随时能拿出来玩，适合文字类操作比较少的游戏，像是逆转裁判。
  2. 性能比安伯尼克强，进入退出游戏基本在2秒左右。

缺点或bug

  1. 电源灯一直亮，熄屏也亮，不太舒服。
  2. 按键松松垮垮的，可能也是故意设置的手感。
  3. 买的复古灰，外壳有点类肤质的感觉，又不太像，有点别扭。
  4. 其他小问题：没双tf卡。没wifi传游戏麻烦，快充的typec线没法充电，只能用低端的线。
  5. 官方系统最新的好像是20240806，8个月没更新固件了，RetroArch还是1.10版，最新的1.20版。

洋葱系统onion os问题

  1. 4.3和4.4一样，耗电很快，纯待机1小时10%左右，电量图标有时不更新，要进游戏再退出才会变。

安伯尼克rg35xxplus

远程传输文件

rg35xxplus连接wifi，记住ip地址。电脑打开ftp软件，例如FileZilla，连接主机填rg35xxplus的ip，例如192.168.1.118
协议sftp（注意不是ftp），端口22，用户名root，密码root，高级-默认本地目录默认远程目录也可以设置好
连接成功后，tf卡2的目录在/mnt/sdcard
注意可能要保持掌机屏幕一直在亮屏状态

开始游戏

安伯尼克rg35xx plus的RetroArch设置更方便的快捷键

rg35xx plus中，系统，设置RetroArch设置，启动RetroArch，是
RetroArch中，设置，输入，快捷键
设置快捷键，按a后再按自己需要设置的键，不设置快捷键，按y，显示—就是没有的快捷键意思
启用快捷键：menu或—，相当于组合键，按住menu再按对应的快捷键才会生效
快进（保持）或快进（开关）：R2
慢放（保持）或慢放（开关）或回溯：L2
加载状态（快速读取）：L1
保存状态（快速存档）：R1
其他没用的都按y取消对应的键，避免冲突
菜单，配置文件，保存当前配置

存档设置

菜单，设置，存档
自动增加状态存储编号，开，作用是每次快速存档，就切换到下一个存档位置，避免错误覆盖
增量状态存储的最大数量：10，或者按自己需要设置，扩展名是.state1-10
自动保存状态：开，扩展名是.state.auto
自动加载存档：开，作用是下次打开游戏，直接载入上次退出游戏的状态（不是自动手动保存的状态）
菜单，配置文件，保存当前配置

RetroArch的mgba核心玩究极绿宝石一开始就卡住

rg35xxplus上，开始游戏前按y选择gpsp核心，就可以玩了。
fix header不知道对正常游戏有没有影响，不建议使用。
fix header只对电脑版的RetroArch有效，rg35xx plus上无效。
下载工具gba tool advance 0.7a，打开究极绿宝石 V-5.gba
header viewer选项卡中，点击fix header，新保存一个xxx.gba的文件就可以玩了

同一个游戏但是不同核心间的存档通用

也适用于高版本转低版本RetroArch
不同核心，即时存档（RetroArch\states）不能通用，要在游戏里存档（RetroArch\saves）
比如精灵宝可梦绿宝石，游戏中，按start，保存，是
如果现在是gba的gpsp核心，那么位置在RetroArch\saves\gpSP的.srm文件
要换成mgba核心，就把这个文件复制到RetroArch\saves\mgba文件夹
如果有的是sav格式的文件，在这个网站选srm-sav转换，https://savefileconverter.com/#/mister
